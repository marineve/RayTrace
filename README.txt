This ray tracer contains all of the desired functionality described in the assignment.
	- It has the ability to render one or more spheres.
	- There is a Floor class, which can be rendered beneath the spheres if desired.
	- All objects in the scene have diffuse shading.
	- It has the ability to light the scene with one or more light sources.
	- It is rendered from a fixed point of view.
	- The objects in the scene cast hard shadows.
	- Simple object-space translations can transform the position of the spheres, instead of modifying their center coordinates.
	- A cube box object is rendered using a triangle mesh.
	- Super-sample antialiasing is used to improve the image.
	- It is possible to create mirrored surfaces.
